# AuroralisOS Site

A concept multi-page website for **AuroralisOS**, a next-generation operating system ecosystem designed by Ethan.

## Pages

- `index.html` — Home, overview of the ecosystem
- `desktop.html` — AuroralisOS Desktop
- `mobile.html` — AuroralisOS Mobile
- `tablet.html` — AuroralisOS Tablet
- `tv.html` — AuroralisOS TV

All pages share the same navigation, styling, and footer.
